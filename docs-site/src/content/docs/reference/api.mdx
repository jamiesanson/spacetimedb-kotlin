The API reference is generated from KDoc comments in the source code using [Dokka](https://kotl.in/dokka).

It covers all public modules:

- **spacetimedb-core** — Client connection, subscriptions, events, and caching
- **spacetimedb-bsatn** — BSATN binary serialization format
- **spacetimedb-codegen** — KSP code generation for tables and reducers
- **spacetimedb-gradle-plugin** — Gradle plugin for module generation

### Building locally

To generate the API docs locally:

```bash
./gradlew dokkaGenerate
```

The output will be at `build/dokka/html/`. Open `index.html` to browse.
